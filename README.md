# D2C Customer Churn Intelligence

## Part 2: RFM Segmentation & Retention Strategy

### Objective

The objective of this project is to identify customer groups requiring retention attention before deploying machine learning models.

This analysis uses RFM (Recency, Frequency, Monetary) segmentation combined with behavioral and support signals to create actionable customer segments.
